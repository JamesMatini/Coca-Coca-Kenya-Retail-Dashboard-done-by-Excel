# Coca-Coca-Kenya-Retail-Dashboard-done-by-Excel
An Excel dashboard analyzed Coca-Cola Kenya's 2022-2023 sales. It revealed sales doubled to ~$24.7M, with top brands Coca-Cola, Dasani, and Diet Coke. Key retailers Amazon, BevCo, and FizzyCo drove growth. Recommendations include strengthening top retailer ties and replicating successful Q3-Q4 strategies in weaker quarters.
